What is Css?

  1. Css stands for Cascading Style Sheets.

  2. Css describes how HTML elements are to be displayed on screen, paper, or in other media.

  3. Css is not neccesaryCSS saves a lot of work. It can control the layout of multiple web pages all at once

  4. External stylesheets are stroed in Css files.

Css can be added to HTML documents in 3 ways:

  • Inline by using the style attribute inside HTML elements
  • Internal-by using a style element in the head section
  • External-by using a link element to link to an external css file
  • The most common way to add css,is to keep the styles in external css file, however in this
    tutorial we will use inline and internal styles,     this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a styles to multiple elements.id, on the other hand, applies a styles to one unique element. ID is also spcial in that you can use a special URL to link directly to an element and its used bt Javascript.

In CSS,selectors are used to target a specific element on a web page. once an element has been targeted,a style can be applied to the element.

There is a wide rang of selectors available, two of the most commonly used are class and ID. both are used to target element to which a style should be applied.
